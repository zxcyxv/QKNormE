# QK-Norm에 의한 커널 해석 전환과 Mean-Shift 변위 벡터

이 페이지는 QK-Norm 적용 여부에 따라 셀프 어텐션의 수학적 성격이 어떻게 달라지며, 그 결과 어텐션 출력의 기하학적 의미가 왜 변하는지를 정리한다.

## 1. 두 가지 커널 체제

### 1A. 표준 트랜스포머 — 내적 커널 (Dot-Product Kernel)

$Q, K$에 정규화를 적용하지 않는 표준 트랜스포머에서 어텐션 스코어는:

$$\text{score}(q_i, k_j) = q_i^\top k_j$$

이 값은 **벡터의 크기(norm)와 방향을 모두 포함하는 unbounded 스칼라**다. 어텐션 가중치는:

$$P_{ij} = \frac{e^{q_i^\top k_j}}{\sum_\ell e^{q_i^\top k_\ell}}$$

이것은 내적 커널(dot-product kernel)이며, 크기에 민감하다. 노름이 큰 키 벡터가 어텐션을 과도하게 흡수할 수 있다.

이 체제에서 $PV$는 "여러 벡터의 가중합"이라는 일반적 의미만 가지며, 특별한 기하학적 구조를 갖지 않는다. Geshkovski (2023)가 분석한 것은 이 체제의 동역학이다 ([[geshkovski2023]]).

### 1B. QK-Norm 트랜스포머 — 가우시안 커널 (Gaussian RBF Kernel)

$Q, K$에 RMSNorm을 적용하면, 정규화된 벡터 $\hat{q}_i, \hat{k}_j \in S^{d-1}$에 대해:

$$\text{score}(\hat{q}_i, \hat{k}_j) = \beta\, \langle \hat{q}_i, \hat{k}_j \rangle$$

단위 초구면 위에서 다음 항등식이 성립한다:

$$\|\hat{q}_i - \hat{k}_j\|^2 = 2 - 2\langle \hat{q}_i, \hat{k}_j \rangle$$

따라서:

$$\langle \hat{q}_i, \hat{k}_j \rangle = 1 - \tfrac{1}{2}\|\hat{q}_i - \hat{k}_j\|^2$$

이를 대입하면:

$$P_{ij} = \frac{e^{\beta\langle \hat{q}_i, \hat{k}_j \rangle}}{\sum_\ell e^{\beta\langle \hat{q}_i, \hat{k}_\ell \rangle}} = \frac{e^{-\frac{\beta}{2}\|\hat{q}_i - \hat{k}_j\|^2}}{\sum_\ell e^{-\frac{\beta}{2}\|\hat{q}_i - \hat{k}_\ell\|^2}}$$

(상수 $e^\beta$는 분자·분모에서 소거)

**이것은 대역폭 $h = 1/\sqrt{\beta}$인 정규화된 가우시안 RBF 커널과 정확히 동일하다.**

이 커널은 이중으로 해석할 수 있다:
- **외재적(extrinsic) 해석**: 구면 위 chordal distance $\|\hat{q} - \hat{k}\|$에 대한 **가우시안 RBF 커널**
- **내재적(intrinsic) 해석**: 구면 위 각도에 대한 **von Mises-Fisher (vMF) 커널** $e^{\beta\langle \hat{q}, \hat{k}\rangle}$

두 표현은 구면 항등식에 의해 행 정규화 후 동치이다.

| | 표준 어텐션 | QK-Norm 어텐션 |
|---|---|---|
| 스코어 | 내적 $q^\top k$ (unbounded) | 코사인 유사도 $\langle\hat{q}, \hat{k}\rangle \in [-1, 1]$ |
| 커널 | 내적 커널 | 가우시안 RBF 커널 |
| 공간 | $\mathbb{R}^d$ (유클리드) | $S^{d-1}$ (초구면) |
| $P$의 성격 | 일반 확률 행렬 | **가우시안 KDE 가중치** |

## 2. 가우시안 커널 체제에서 $PV$의 의미 전환

### 2A. $PV$는 Barycentric Target (커널 가중 평균)

$P$가 가우시안 커널 가중치일 때, $PV = \sum_j P_{ij} V_j$는:

- 주변 토큰 $\{V_j\}$의 **가우시안 가중 평균** (볼록 결합)
- 커널 밀도 추정(KDE) $\hat{p}(x) = \sum_j K(x, x_j)$의 **Nadaraya-Watson 추정치**
- 밀도 극대점(mode) 자체가 아니라, **모드로 향하는 한 스텝의 barycentric target** (fixed-point iteration의 다음 반복값)

즉, $PV$는 "현재 토큰이 이동해야 할 목적지"이지, "이동해야 할 방향과 크기(변위)"가 아니다. 다만 이 "목적지"는 밀도의 극대점 그 자체가 아니라, 극대점을 향한 **한 스텝의 중간 지점**이라는 점을 정밀히 구분해야 한다.

### 2B. 잔차 연결에서의 기하학적 비정합성

표준 잔차 연결:

$$X^{[k+1]} = X^{[k]} + PV$$

이것은 "현재 좌표 + 목적지 좌표"로, 구면 기하 해석을 채택하면 비정합적이다.

> **주의: "모순"인가 "비최적"인가?**
>
> 이 비정합성을 "형식적 모순"이라 부르는 것은 과하다. 실제 트랜스포머의 residual stream은 $\mathbb{R}^d$의 벡터공간이며, 그 안에서는 모든 벡터가 동시에 "위치"이자 "변위"로 해석 가능하다. 더 정확한 표현은 **"구면 기하 해석을 채택하면 non-canonical"**이다. 상세 논의는 [[PV의_기하학적_적합성]] 참조.

## 3. Mean-Shift와 $(P-I)V$의 도출

### 3A. Mean-Shift 알고리즘

[[rigollet2024]]의 Section 5.3에서 명시적으로 증명된 바:

구면 위의 가우시안 커널 blurring Mean-Shift 동역학은

$$\dot{x}_i(t) = \nabla \log (K * \mu_t)(x_i(t))$$

이며, 이것은 SA 동역학과 **정확히 일치**한다.

Mean-Shift에서 밀도 극대점을 향한 그래디언트 어센트 방향, 즉 **mean-shift 벡터**는:

$$m(x_i) = \frac{\sum_j K(x_i, x_j)\, x_j}{\sum_j K(x_i, x_j)} - x_i = (PV)_i - V_i$$

이것이 **변위 벡터(displacement vector)**이며, 잔차 연결에 더해져야 하는 올바른 양이다:

$$X^{[k+1]} = X^{[k]} + (PV - V) = X^{[k]} + (P - I)V$$

### 3B. 도출의 논리 체인

1. QK-Norm 적용 → 어텐션이 가우시안 커널로 전환
2. 가우시안 커널 어텐션 = Mean-Shift 탐색
3. Mean-Shift의 이동 방향 = $\nabla \log \hat{p}(x) \propto PV - V$
4. 따라서 잔차에 더할 올바른 양은 $PV$가 아니라 $(P-I)V$

### 3C. $(P-I)V$의 한계와 주의사항

위 도출은 $H = V$, $W_O = I$인 이상화에서 성립한다. 실제 트랜스포머에서는 다음 문제가 있다:

1. **좌표계 불일치**: 잔차 업데이트의 "현재점"은 $h_i$인데, $(P-I)V$가 빼는 것은 $v_i = h_i W_V$이다. 일반적으로 $v_i W_O \neq h_i$이므로, 올바른 "target minus current"는 $(PVW_O)_i - h_i$이지 $((P-I)VW_O)_i$가 아니다.
2. **Identity head 무효화**: $P \approx I$인 head에서 $(P-I)V \approx 0$이 되어, 자기 정보 통과 기능이 사라진다.
3. **Pre-RMSNorm에서의 유클리드 공간**: Pre-RMSNorm 구조에서 residual stream $H$는 초구면이 아닌 자유로운 유클리드 공간의 벡터이다. 따라서 접선사영 논의 자체가 불필요하며, 유클리드 공간에서의 mean-shift displacement는 단순히 $PVW_O - H$이다.

이 한계들에 대한 상세 분석과 대안은 [[잔차연결_수정안]] 참조.

## 4. 그래프 라플라시안 해석

$(P - I)V$를 부호를 바꿔 쓰면:

$$-(I - P)V = -L_{rw}\, V$$

여기서 $L_{rw} = I - P$는 **정규화된 랜덤 워크 그래프 라플라시안**이다.

### 그래프 라플라시안의 성질

- $L_{rw}$의 고유값: $0 = \lambda_1 \leq \lambda_2 \leq \cdots \leq \lambda_n$
- 상수 벡터 $\mathbf{1}$에 대한 고유값이 0 → **전역 평균은 보존**
- $L_{rw}V$는 그래프 위의 확산(diffusion) 연산

### 확산 방정식으로서의 해석

$$\frac{\partial V}{\partial t} = -(I - P)V = (P - I)V$$

이것은 그래프 위의 **비등방성 확산 방정식(anisotropic diffusion equation)**:

- **노이즈 억제:** 유사한 토큰 사이의 차이를 감소 (가우시안 가중치 $P_{ij}$가 가까운 토큰에 집중)
- **의미론적 경계 보존:** 서로 다른 의미 클러스터 사이의 차이는 유지 (가우시안 가중치 $P_{ij}$가 먼 토큰에 거의 0)
- **Perona-Malik 확산**과의 유사성: 이미지 처리에서의 edge-preserving smoothing과 동일한 원리

### Geshkovski (2023)의 리스케일 동역학과의 비교

[[geshkovski2023_appendix_PIV]]에서 정리한 바와 같이, Geshkovski의 리스케일 동역학(식 3.1) 역시 $(P-I)V$ 형태를 가진다:

$$\dot{z}_i = \sum_j P_{ij}\, V(z_j - z_i)$$

그러나 두 맥락은 다르다:

| | Geshkovski (2023) 리스케일 | 본 해석 (QK-Norm 체제) |
|---|---|---|
| 등장 경위 | 분석 편의를 위한 변수 치환 $z = e^{-tV}x$ | 커널 전환에 의한 기하학적 필연성 |
| 전제 | $\mathbb{R}^d$, 정규화 없음, 일반 $V$ | $S^{d-1}$, QK-Norm, 가우시안 커널 |
| 의미 | 수학적 기법 | **아키텍처 설계 원리** |
| 근거 | 토큰 노름 발산 제어 | Mean-Shift 변위 벡터의 정의 |

## 5. 요약: 논리 체인

```
QK-Norm (RMSNorm on Q, K)
    ↓ Q, K → S^{d-1}로 사영
내적 커널 → 가우시안 RBF 커널로 전환
    ↓ ||q-k||² = 2 - 2⟨q,k⟩
P = 가우시안 KDE 가중치
    ↓ Rigollet (2024) Section 5.3
PV = 밀도 극대점의 좌표 (Mean-Shift target)
    ↓ Mean-Shift vector = target - current
(P-I)V = 변위 벡터 (displacement)
    ↓ -(I-P) = -L_rw
그래프 라플라시안에 의한 비등방성 확산
```
